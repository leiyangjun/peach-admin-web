# peach-admin-web

Peach 管理端前端工程，基于 **Vue 3 + Vite + TypeScript + Element Plus + Pinia**，与 `peach-gateway`、后端微服务联调。

## 技术栈

- Vue 3、Vue Router 4
- Vite 8
- TypeScript
- Element Plus
- Pinia
- Axios

## 常用命令

```bash
cd peach-admin-web
npm install
npm run dev
npm run build
npm run preview
```

## 构建产物

- 生产构建输出目录：`dist/`（已在 `.gitignore` 中忽略，勿提交）

## 远程仓库与双推送

已配置 `origin` 同时推送 **GitHub** 与 **Gitee**，一次推送即可同步：

```bash
git push -u origin main
```
